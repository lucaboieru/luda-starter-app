# luda-starter-app
Starter app for Luda node framework

### Install Luda Framework
```git clone https://github.com/square-gmbh/luda.git```

### Install node modules
```npm install```

### Run the server
```node luda/server.js <PORT>```

### Example
```node luda/server.js 9999```
